# Icon Concepts

All concepts are black-only SVG marks on transparent backgrounds.

- `01-aperture-delta.svg`: current README icon. Circle = field boundary, broken arcs = bidi reference transfer, triangle = delta, opposing arrows = bidirectional coupling.
- `02-nested-boundary.svg`: more software/system-like. Strong nested module boundary, less mathematical.
- `03-trit-barrier.svg`: more dynamics/control-like. Emphasizes trit barrier and invariant floor.
- `04-dual-cone.svg`: more formal/math-like. Diamond field with up/down and left/right coupling.

The primary front-page file is `../bidi-cdc-icon.svg`.
